 10   
 11      A GraphESN performs following operations: 
 12   
 13          Input -> Encoding -> Mapping -> Readout -> Output Activation 
 14   
 15      See details below. 
 16   
 17      GraphESN can use memoization, avoiding the reprocessing of previously 
 18      evaluated inputs. 
 19   
 20      Attributes: 
 21      ----------- 
 22      Nu      --  Input label dimension. (int) 
 23      Nr      --  Reservoir size, i.e. number of units. (int) 
 24      Ny      --  Output size. (int) 
 25      W_in    --  Input-to-reservoir matrix (scipy.array (Nr,Nu+1)) 
 26      W       --  Reservoir-to-reservoir matrix (scipy.array (Nr,Nr)) 
 27      W_out   --  Reservoir-to-readout matrix (scipy.array (Ny,Nr+1)) 
 28      map_fun --  State mapping function: (|V(g)|,Nr) -> (Nr,) 
 29      out_act --  Output activation function: (Ny,) -> (Ny,) 
 30      epsilon --  Reservoir convergence threshold (float). 
 31      maxit   --  Maximum number of iterations allowed for the reservoir (int). 
 32   
 33   
 34      Main methods: 
 35      ------------- 
 36      encode          --  Get the internal representation of a Graph. 
 37                          (list of scipy.array (Nr,)) 
 38      get_activations --  Get the reservoir activations after the state mapping 
 39                          function has been applied. (scipy.array (Nr,)) 
 40      evaluate        --  Get the output for an input graph. (scipy.array (Ny,)) 
 41   
 42   
 43      Memoizing: 
 44      ---------- 
 45      memo_enc    --  Save results for the 'encode' function (boolean). 
 46      memo_act    --  Save results for the 'get_activations' function (boolean). 
 47      memo_eval   --  Save results for the 'evaluate' function (boolean). 
 48   
 49      All the values are True by default. 
 50   
 51      ---------------------------------------------------------------------------- 
 52   
 53      Input 
 54      ------------------------ 
 55      A (directed) Graph 'g = (V(g), E(g))'. Each vertex has an attached label 
 56      (scipy.array) in its attributes. 
 57   
 58   
 59      Input -> Encoding  
 60      ------------------------ 
 61      Input graph is encoded into a vector. Internal representation is 
 62   
 63          x(g) = scipy.array (Nr * |V(g)|,).  
 64   
 65      See method 'encode'. 
 66   
 67   
 68      Encoding -> Mapping 
 69      ------------------------ 
 70      Obtained encoding is mapped into a fixed length vector in order to 
 71      achieve a size independent representation. 
 72      A state mapping function, X : (Nr * |V(g)|,) -> (Nr,) is applied  
 73      e.g. 'mean_state_mapping' and 'supersource_state_mapping'. 
 74      See method 'get_activations'. 
 75   
 76   
 77      Mapping -> Readout 
 78      ------------------------ 
 79      Reservoir activations returned by the mapping function are sent to network's 
 80      readout: a linear combination is computed. Weights are stored in the 
 81      matrix W_out (scipy.array (Ny, (Nr + 1))) 
 82   
 83          net = W_out * X(x(g)), scipy.array (Ny,) 
 84       
 85   
 86      Readout -> Output Activation 
 87      ------------------------ 
 88      Linear combination result is used as input of the output activation  
 89      function, which is applied elementwise. 
 90   
 91          out = f(net) = f(W_out * X(x(g))) 
 92   
 93      In order to train the network, the output activation function can be   
 94      assumed to be derivable (e.g. 'g_esn.functions.tanh'). Thus it's recommended 
 95      to use an ActivationFunction object (see package g_esn.functions). 
 96      Output activation is the actual output of the network. 
 97      See method 'evaluate'.
