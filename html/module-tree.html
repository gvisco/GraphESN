Module Hierarchy

  • g_esn: Implement basic functionalities of GraphESN.
    • g_esn.constructive: Implement constructive models based on GraphESN.

      • g_esn.constructive.pluggable_phi_graph_esn: Implementation of the constructive PhiGraphESN model.
    • g_esn.functions: Common functions used by GraphESN and learning procedures.
    • g_esn.graph
    • g_esn.graph_esn: Module defines the class GraphESN, implementing the standard Echo State Network for graphs.
    • g_esn.learning: Provide functions for models trainining.
    • g_esn.parsers: Parsers: from file to g_esn.graph.Graph lists.
      • g_esn.parsers.gph_parser: Module provides a Python representation of a GPH-standard file.
      • g_esn.parsers.pl_parser
      • g_esn.parsers.sdf_parser
    • g_esn.phi_graph_esn: Phi-GraphESN implementation.
    • g_esn.utils: Package contains modules that ease the use, the manipulation and the analysis of models, data, results.
